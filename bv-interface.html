This project belongs to a fellow Bazaarvoice intern, Andee Liao, who was assigned to create a UI to explore Bazaarvoice's EMOdb's content and functionality, which was previously only accessable through command line. My role was to create the design for the interface and implement it via HTML, CSS, and JS.

Click here to view this project on GitHub.
After Andee finished all of the functionality and back-end aspects of the project, she presented it to me as follows. The structure was very bare bones, with little to no CSS, other than the functional necessities. My role was to take this and design something with better user flow and design elements.

Use cases: The primary users would be software engineers who are familiar with the EMOdb functions. Thus, the interface does not need much explanatory elements, but should ensure a simple and intuitive flow of information.
